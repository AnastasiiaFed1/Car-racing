# Car Racing Game (Pygame)

## Опис проєкту

Це 2D-гра у жанрі аркадних перегонів, реалізована з використанням бібліотеки Pygame.
Гравець керує автомобілем, уникаючи зіткнень з іншими машинами та дорожніми перешкодами (конусами).

Гра побудована за модульною архітектурою з розділенням логіки на окремі компоненти: гравець, перешкоди, дорога, UI та керування станами гри.

## Геймплей

* Керування автомобілем здійснюється стрілками
* Перешкоди з’являються випадково на смугах дороги
* Швидкість гри поступово зростає
* За кожну уникнену перешкоду нараховуються очки
* При зіткненні гра завершується
  
## Основні можливості

* Рух гравця з обмеженням по межах дороги
* Генерація різних типів перешкод (машини, конуси)
* Система колізій
* Система очок і рівнів
* Динамічна дорога з розміткою
* Інтерфейс користувача (меню, game over, HUD)

## Структура проєкту

```
project/
│
├── main.py              # Основний цикл гри
├── player.py            # Логіка гравця
├── obstacles.py         # Перешкоди та їх генерація
├── settings.py          # Налаштування гри
│
└── game/
    ├── road.py          # Відображення дороги
    ├── ui.py            # Інтерфейс користувача
    └── state.py         # Стани гри
```

## Встановлення та запуск

### 1. Встановити залежності

```bash
pip install pygame
```

### 2. Запустити гру

```bash
python main.py
```

## Керування

* LEFT/RIGHT рух автомобіля
* ENTER — почати гру
* ESC — вихід
* R — рестарт після програшу
* M — повернення в меню

## Особливості реалізації

* Використано об'єктно-орієнтований підхід (класи для кожного компонента)
* Реалізовано систему спавну перешкод через окремий клас `Spawner`
* Логіка гри відокремлена від відображення
* Взаємодія між модулями реалізована через чіткі інтерфейси
